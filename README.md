# Converter
## Number converter

Converter is a program that allows you to pass a number from the decimal system to the binary and from the binary to the decimal


## Installation

```
npm install convert
```

## Use

```
import { decimalTobinary } from 'convert'
import { binaryTodecimal } from 'convert'

```
